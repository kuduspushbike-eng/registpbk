Halo! Sesuai permintaan Anda, sekarang pendaftaran untuk **Member Lama** telah dipermudah sepenuhnya secara otomatis:

### Apa Saja Perubahannya?
1. **Tanpa Biaya & Tanpa Konfirmasi**: Ketika pendaftar memilih **"MEMBER LAMA"** (baik yang otomatis terdeteksi oleh sistem maupun yang memilih tombol klaim member lama manual), mereka **tidak akan ditagih biaya apa pun (Gratis)**.
2. **Tanpa Approval Admin**: Mereka **tidak perlu menunggu persetujuan (approval/waiting approval)** lagi! Setelah memasukkan nomor WA dan nama, mereka bisa langsung mengisi data lengkap di formulir.
3. **Tanpa Pilihan Ukuran Baju**: Karena member lama tidak mendapat kaos, pilihan ukuran baju **otomatis dihilangkan** dari formulir (baik untuk 1 anak maupun 2 anak). Pada spreadsheet, kolom ukuran baju otomatis terisi strip `"-"`.
4. **Langsung Gabung Grup WhatsApp**: Setelah selesai mengisi data pendaftaran, mereka langsung diarahkan ke halaman sukses dengan tombol hijau besar untuk **"Gabung Grup WhatsApp"**.

---

### Langkah Update Google Apps Script (Sangat Penting!):
Agar perubahan sistem ini berjalan dengan lancar, silakan perbarui kode di Google Apps Script Anda ke versi **v15**:

1. Buka halaman **Google Apps Script** Anda.
2. Hapus seluruh isi file `Code.gs` yang lama.
3. Salin (**Copy**) seluruh isi kode dari file `Code.gs` baru di editor sebelah kiri, lalu tempel (**Paste**) ke Apps Script Anda.
4. Klik tombol **Save** (ikon disket).
5. Lakukan Deploy ulang:
   - Klik **Deploy** di pojok kanan atas -> Pilih **Manage deployments**.
   - Klik ikon **Pensil (Edit)** pada deployment aktif.
   - Pada bagian **Version**, pilih **New version**.
   - Klik tombol **Deploy** di pojok kanan bawah.

Silakan langsung dicoba pendaftarannya ya! 👍
